## Prerequisites

This project requires Java >= 21 to be installed.

## Setup of project

1. Clone the repository:

   ```bash
   git clone https://github.com/Luxcorel/project-course-2
   ```

2. Navigate to the project directory:

   ```bash
   cd project-course-2
   ```

3. Build and run the project using Gradle:

   MacOS/Unix:
   ```bash
   ./gradlew run
   ```
   
   Windows:
   ```bash
   gradlew run
   ```
